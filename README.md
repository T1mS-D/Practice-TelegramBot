# Агрегатор переписок в Telegram для обработки большого потока писем и поиск писем по промпту

## Описание проекта
Система для сбора и семантического анализа сообщений из Telegram-каналов и групп.
Проект решает задачу мониторинга больших потоков информации: пользователь описывает "промпт", а бот автоматически находит релевантные сообщения и присылает уведомление со ссылкой на оригинал.

Ключевая особенность: семантический поиск по тексту, что позволяет отслеживать не просто ключевые слова, а контекст (сроки, обязательства, упоминания брендов).

## Целевая аудитория
- Бизнес-пользователи.
- Исследователи и аналитики.
- Команды, работающие с большим количеством Telegram-источников.
